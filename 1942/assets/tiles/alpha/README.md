# Tech Architect Alpha — Tile Set

**Approach:** CLEAN & GEOMETRIC  
**Campaign:** Coral Front (Level 1)  
**Date:** 2026-03-02  

## Philosophy

Sharp, clean pixel art with clear geometric shapes and high contrast. All tiles are designed at **16×16 native resolution** and scaled **4× to 64×64** using nearest-neighbor interpolation, giving the chunky retro look specified in the tile system spec.

## Grid Layout

- Main body offset rightward

- Cove indent on the east side
- Irregular south coastline with a spit
- Deep → shallow → sand → grass layering

## Two-Track Generation Approach

### Track 1: Programmatic (clean/)
- Generated by `generate-clean-tiles-v2.py`
- Pixel-perfect at native 16×16
- **Guaranteed** edge color matching between tiles
- Consistent dither patterns across the set
- Best for: reliable tileability, game integration

### Track 2: AI-Enhanced (enhanced/)
- Original nano-banana-pro tiles in root directory (1K resolution)
- Enhanced versions in `enhanced/` (fed clean tiles → Gemini for refinement)
- Richer visual detail but inconsistent edge colors
- Best for: visual reference, inspiration for hand-tuned tiles

### Verdict
The **clean programmatic set** is the recommended base for game integration. Select AI-enhanced tiles can be used after manual edge-matching to add visual richness where needed. The `water-deep` and `sand` enhanced versions are strong candidates for this hybrid approach.

## Known Limitations

1. **Grid-bound shapes** — Island silhouettes are inherently rectangular due to tile grid constraints. Diagonal/curved transition tiles would improve organic feel.
2. **No east/west sand-grass transitions** — Only north/south transitions are implemented. Adding edge-E/W grass variants would allow more organic grass placement.
3. **Cloud tiles have opaque backgrounds** — Should ideally use transparent PNGs for overlay compositing. The dark blue sky background is a workaround.
4. **Single variant per tile type** — Adding 2-3 randomized variants per base tile would break visible repetition patterns.
5. **Missing tile types** — terrain-rock (#12), terrain-structure (#13), terrain-path (#14) are not yet implemented.
